	 - 방향키 좌(←): 좌로 이동
	 - 방향키 우(→): 우로 이동  
	 - 'a'/'A': 반시계방향 회전
	 - 's'/'S': 시계 방향 회전
	 - spacebar: 중앙의 메인 룰렛 회전
